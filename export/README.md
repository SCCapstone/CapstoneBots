# BVCS Blender Add-on Installation Guide

Follow these quick steps to install and enable the **Blender Version Control System (BVCS)** add-on.

---

## 1. Prepare the Add-on ZIP

1. Find the `blender_vcs` folder in the project.
2. **Zip the folder** — the ZIP must contain a folder named `blender_vcs` with `__init__.py` inside.

**Correct ZIP structure:**

```
blender_vcs.zip
└── blender_vcs/
    └── __init__.py
    └── (any other file in this folder)
```

---

## 2. Move ZIP to Export Folder (Optional)

Place the ZIP file into your `export` folder to keep things organized.

---

## 3. Open Blender Preferences

1. Open Blender.
2. Go to **Edit → Preferences**.

   * Shortcut: **Ctrl + ,** on Windows (Mac: **Cmd + ,**)

---

## 4. Install the Add-on

1. Navigate to the **Add-ons** tab.
2. Click the **down arrow** in the top-right corner.
3. Select **Install from Disk**.
4. Find and select your ZIP file.
5. Click **Install Add-on**.

---

## 5. Enable the Add-on

Blender should auto-enable it.
If not, check the box next to **BVCS** to enable it manually.

---

## 6. Access the Add-on

1. In the 3D Viewport, press **N** to open the right-hand sidebar.
2. Click the **BVCS** tab.

---

## Done!

Your BVCS add-on is now installed and ready to use. Enjoy!
